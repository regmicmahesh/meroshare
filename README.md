```
NAME:
   meroshare-cli - meroshare-cli is a command line interface for Meroshare.

USAGE:
   meroshare-cli [global options] command [command options] [arguments...]

DESCRIPTION:
   Meroshare CLI - Reject Angular and Bloated Web and embrace the power of CLI.

COMMANDS:
   details    Get user details
   portfolio  Get Portfolio
   init       Initialize credentials for Meroshare.
   help, h    Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --output value  output in json or ascii table (default: "ascii")
   --help, -h      show help (default: false)
```
